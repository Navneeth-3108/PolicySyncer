# Network Security Policy

**Author:** Noah Patel

**Department:** Operations

**Version:** v3.0

**Last Reviewed:** 2022-12-15

**Status:** active

- All users must access as per company standards.
- All users required change as per company standards.
- Patch is prohibited for all users.
- Monitoring is prohibited for all users. (Reference: NIST SP 800-53 Rev 4)
- All users required endpoint as per company standards. (Reference: WEP)
- All users must third-party as per company standards.
- All users recommended third-party as per company standards.
- All users must mobile as per company standards.
- All users recommended endpoint as per company standards.
